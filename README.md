# streetmap

projet de prise en main de la bibliothèque leafletJs.

## run

 - `git clone`
 - `docker-compose up`

## stop

 - `docker-compose down`

## fonctionalitées

 - map : menu burger en haut à droite de la map
 - admin : voir les examples
